# Описание функционала файла DZ_VD01.html

## Общая информация

**Файл:** `DZ_VD01.html`  
**Название страницы:** "Любимые блюда"  
**Назначение:** HTML-страница с таблицей рейтинга любимых блюд

## Структура документа

Файл представляет собой полноценный HTML-документ со стандартной структурой:

- **DOCTYPE** - объявление типа документа HTML5
- **Тег `<html>`** - корневой элемент с атрибутом `lang="en"`
- **Тег `<head>`** - секция метаданных:
  - Кодировка UTF-8 для поддержки русского языка
  - Заголовок страницы "Любимые блюда"
- **Тег `<body>`** - основное содержимое страницы

## Основной функционал

### Таблица рейтинга блюд

Страница содержит одну таблицу (`<table>`), которая отображает рейтинг из 5 блюд.

#### Структура таблицы

**Заголовки столбцов (`<th>`):**
1. Название блюда
2. Оценка
3. Ссылка на рецепт
4. Фото

**Строки данных (`<tr>` с `<td>`):**
Таблица содержит 5 строк с информацией о блюдах:

1. **БОРЩ** - оценка 10/10
2. **ПЛОВ С КУРИЦЕЙ** - оценка 8/10
3. **КОТЛЕТЫ ИЗ ФАРША КЛАССИЧЕСКИЕ** - оценка 9/10
4. **СУП С КУРИНЫМИ СЕРДЕЧКАМИ ЛАПШОЙ И КАРТОШКОЙ** - оценка 10/10
5. **ШАРЛОТКА С ЯБЛОКАМИ** - оценка 10/10

### Используемые HTML-элементы и атрибуты

#### Тег `<table>`
- **`bgcolor="#98fb98"`** - устанавливает светло-зеленый цвет фона таблицы
- **`align="center"`** - выравнивание таблицы по центру
- **`border="5px"`** - устанавливает рамку толщиной 5 пикселей вокруг таблицы
- **`height="400px"`** - фиксированная высота таблицы 400 пикселей
- **`width="800px"`** - фиксированная ширина таблицы 800 пикселей

#### Тег `<a>` (гиперссылки)
- **`href`** - абсолютные ссылки на рецепты с сайта 1000.menu
- **`target="_blank"`** - открытие ссылок в новой вкладке браузера

#### Тег `<img>` (изображения)
- **`src="/test/..."`** - относительные пути к изображениям в папке `test`
- **`width="200"`** - ширина изображений 200 пикселей (высота устанавливается автоматически пропорционально)
- **`alt`** - альтернативный текст для изображений (важен для доступности и SEO)
- **`title`** - всплывающая подсказка при наведении курсора на изображение

### Используемые изображения

Все изображения находятся в папке `/test/`:
- `борщ.jpg`
- `плов.jpg`
- `котлеты.jpg`
- `суп_лапша.jpg`
- `шарлотка.jpg`

### Внешние ссылки

Все ссылки ведут на сайт 1000.menu с конкретными рецептами:
- Классический борщ с мясом
- Вкусный плов с курицей в мультиварке
- Котлеты из фарша классические на сковороде
- Суп с куриными сердечками лапшой и картошкой
- Простая шарлотка
